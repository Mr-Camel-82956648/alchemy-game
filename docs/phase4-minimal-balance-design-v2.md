# 《AI法阵·炼金术士》最小数值策划案 v2

> 状态：已确认，作为 Phase 4 数值与战斗收口的实现基线  
> 范围：最小可落地版本，不包含后续扩展系统

---

## 1. 设计目标

《AI法阵·炼金术士》的核心不是传统数值构筑，而是：

- AI 提供视觉上无限丰富的大招生成可能
- 炼金系统负责创造新法阵
- 割草战斗负责承载炼金等待期，并为炼金供能
- 数值系统负责把“视觉差异”翻译成“玩法差异”

本阶段数值设计遵循以下原则：

1. **效果优先，数值支撑**
   - 优先让玩家看得懂、感受得到，再做复杂公式。

2. **规则少，但反馈要明显**
   - 不追求复杂 build。
   - 追求“打对了很爽，打错了很痛”。

3. **视觉差异必须映射成战斗差异**
   - 不同法阵不能只换名字和视频，必须在战斗表现上有区别。

4. **优先低美术成本实现高反馈规则**
   - 尽量通过 scale、颜色、血条、HUD、怪物行为参数实现差异。
   - 不依赖新增怪物技能或大量新增素材。

---

## 2. 玩家主循环

游戏的最小主循环为：

1. 玩家选择两张法阵进行炼金合成
2. 炼金开始，法阵生成进入等待阶段
3. 玩家进入割草战斗，在时限内击杀敌人收集亡魂
4. 亡魂为炼金炉供能
5. 战斗结束后进入以下结局之一：
   - **成功**：在时限内收集足够 Soul，且玩家存活，炼金完成
   - **失败 A**：时限结束但 Soul 不足，炼金失败
   - **失败 B**：玩家死亡，炼金失败
6. 回到炼金室领取成果 / 调整装备 / 再次合成

### 结局文案

- 开始前：
  - **「炉中之火，以魂为薪」在时限内收割足够的忘魂，方能炉火不熄。**
- 成功：
  - **「千魂融沸，炉火长明」亡魂悉数归炉，炼金已成，灵火不灭。**
- 失败（Soul 不足）：
  - **「时限已至，炼金失败」未能收割足够的亡魂，炉火已于黑暗中孤寂熄灭。**
- 失败（死亡）：
  - **「血肉崩解，魂归荒野」你倒在了亡灵的围攻之下，未能抵挡住狂潮的侵蚀。**

## 3. 法阵规则

### 3.1 法阵的核心战斗字段

每张法阵至少包含以下字段：

- `generation`
- `mainAttr`
- `subAttr`
- `baseAtk`
- `name`
- `videoUrl`
- `prompt / visualDesc`

其中，本阶段真正参与核心战斗规则的是：

1. **主属性 `mainAttr`**
2. **代数 `generation`**
3. **基础攻击 `baseAtk`**

副属性 `subAttr` 保留，但第一阶段仅轻度参与。

### 3.2 四属性体系

四大属性及视觉识别建议如下：

- **火**：赤红 / 橙红
- **冰**：冰蓝 / 白
- **雷**：亮金 / 黄
- **蚀**：墨绿 / 黑

### 主属性作用

主属性决定：

- 怪物抗性判定
- 玩家打对/打错属性时的结果
- 法阵的第一层战斗差异

### 3.3 代数成长

代数用于鼓励持续合成。  
代数越高，法阵基础强度越高，但不能强到无视属性克制。

建议公式：

```txt
baseAtk = 100 + (generation - 1) * 30
```

示例：

- Gen01 = 100
- Gen02 = 130
- Gen03 = 160
- Gen04 = 190
- Gen05 = 220

### 新法阵代数

```txt
newGeneration = max(parentA.generation, parentB.generation) + 1
```

### 3.4 副属性定位

副属性保留，原因是：

- 强化“融合”感
- 鼓励玩家天马行空地尝试合成
- 使法阵不只是单元素概念

但本阶段不让副属性承担过重规则职责。

### 本阶段副属性用途

- 用于展示
- 用于法阵风格解释
- 为后续扩展保留接口

### 本阶段不做

- 不做复杂双属性乘算
- 不做副属性主导的克制判定
- 不做大量由副属性触发的特殊效果

## 4. 怪物规则

怪物差异分为两层：

1. **属性层**
2. **行为层**

### 4.1 属性层

#### 普通怪

- 拥有 1 个抗性属性

#### 精英 / 缝合怪

- 拥有 2 个抗性属性
- 体型更大
- 血量更厚
- 视觉上应明显具有拼接感 / 双属性感

### 4.2 行为层

由于怪物不做复杂弹幕与技能，本项目中怪物差异主要通过以下行为参数实现：

- `speed`：速度快慢
- `movePattern`：运动轨迹
- `groupPattern`：群体形状
- `hpScale`：生命系数
- `sizeScale`：体型系数

### 推荐的移动模式

- `direct`：直线追踪
- `sway`：轻微摆动接近
- `arc`：弧线包抄
- `burst`：短停顿后突进

### 推荐的群体形状

- `cluster`：团簇
- `line`：横排推进
- `wedge`：楔形 / 矛头
- `ringLoose`：松散围拢

---

## 5. 属性与行为的统一原则

怪物应尽量做到“看起来像什么，打起来也像什么”。

建议弱绑定如下：

- **火系怪**
  - 更直接
  - 中速或偏快
  - 更有正面压迫感

- **冰系怪**
  - 偏慢
  - 更厚重
  - 更适合成排推进

- **雷系怪**
  - 更快
  - 更敏捷
  - 更适合突进、变向

- **蚀系怪**
  - 更飘忽
  - 更适合绕行、包抄
  - 更适合松散围拢

## 6. 核心战斗规则

### 6.1 非抗性命中

当玩家法阵主属性不在怪物抗性列表中时：

- 正常造成伤害

### 6.2 抗性命中（核心特色规则）

当玩家法阵主属性命中怪物抗性属性时：

- 怪物**不掉血**
- 怪物**吸收该攻击**
- 怪物**scale 增大**
- 怪物**最大生命提高**
- 怪物**当前生命同步提高**
- 该效果可叠层，但应设上限（建议 2\~3 层）

### 设计目的

此规则用于强化以下体验：

- 打错属性不是“无事发生”，而是“喂怪”
- 玩家必须关心自己装备的法阵属性
- 怪物属性抗性具备鲜明视觉后果
- 避免“看起来差很多，实际打起来一样”的问题

### 示例规则

每次吸收后：

- `maxHp += baseMaxHp * 0.15`
- `hp += baseMaxHp * 0.15`
- `scale += 0.08`

上限：

- 最多叠加 2\~3 次

## 7. 波次与战斗目标

本游戏不强调关卡制，也不强依赖固定 Boss。  
核心是“一次次轮回式的炼金战斗”。

### 7.1 波次逻辑

- 游戏以 wave 形式逐步增加敌人压力
- 7\~10 种怪物应逐渐登场
- 类似植物大战僵尸的“玩几局后出现新敌人”体验

### 7.2 当前阶段不强调

- 独一无二的大 Boss
- Boss 独立技能体系
- 复杂阶段演出

### 7.3 精英替代 Boss

当前阶段可通过以下方式承担“中高威胁单位”职责：

- 双抗缝合怪
- 更大体型
- 更高血量
- 更极端的行为参数

## 8. 战斗中的三轴资源

战斗围绕以下三项资源展开：

### 8.1 HP

- 玩家生存条件

### 8.2 Soul

- 炼金供能资源
- 击杀敌人后，亡魂应向坩埚 HUD 聚拢
- 当前 soul 信息条未来可演化为围绕坩埚图标的圆环统计图

### 8.3 Time

- 炼金时限
- 到时进行结局判定

## 9. 卡牌 UI 呈现原则

遵循“视觉优先，少字少数值”的原则。

### 建议显示

- 卡牌名称
- `GenXX` 角标
- 主属性 icon
- 副属性 icon（如有）

### 不建议显示

- 大段属性文字
- 直接裸露的攻击力数字
- 冗长说明文案

### 原因

本游戏不是传统数值装备游戏，  
玩家需要感知的是：

- 这是第几代法阵
- 它偏什么属性
- 它大致适合打什么怪

而不是盯着一串攻击数字。

## 10. 旧卡与正式卡策略

### 10.1 旧测试卡

当前收藏中的旧测试卡：

- 视觉效果不达标
- 最终将删除
- 不作为长期正式内容维护对象

因此：

- 不需要为旧卡设计完整长期兼容方案
- 当前仅需最低限度保证不报错

## 11. 第一阶段实现范围

本阶段只做最小必要实现，目标是让“视觉差异 → 战斗差异”成立。

### 必做项

1. 把 `generation / baseAtk / mainAttr` 真正接入战斗结算
2. 普通怪实现单抗
3. 精英 / 缝合怪实现双抗
4. 实现“抗性吸收 → 变大 + 血量变厚”机制
5. 增加怪物行为参数差异：
   - 快慢
   - 轨迹
   - group 形状
6. 战斗目标统一为 HP / Soul / Time 三轴
7. 卡牌 UI 最小显示：
   - 名称
   - Gen
   - 属性 icon

### 可后置项

1. 副属性深度玩法
2. 真正的 Boss 与独特技能
3. 更复杂的成长系统
4. 商店、资源货币、图鉴奖励
5. 完整旧卡清理工具
6. 更复杂的法阵特效差异化

## 12. 本阶段的成功标准

如果本阶段完成后，玩家能明显感受到以下几点，即视为成功：

1. 不同法阵打不同怪，结果明显不同
2. 打错属性会出现明确且可见的负反馈
3. 合成更高代法阵能感受到稳定成长
4. 缝合怪/精英怪不靠新技能，也能形成新的压力体验
5. 玩家会开始主动考虑：
   - 我该带什么法阵上场
   - 这一波怪偏什么属性
   - 我要不要继续合成更高代的法阵
