# 历史文档归档说明

本目录用于存放仍有参考价值、但不再代表当前代码与当前机制真相的历史文档。

使用原则：

- 需要了解旧阶段目标、旧交接语境或旧设计来源时再查看

- 当前有效入口请回到根目录 `README.md`，并优先阅读 `docs/attrset-quota-walkthrough.md` 和 `docs/forge-schema.md`
